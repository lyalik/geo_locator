# 🏆 Geo Locator - Система детекции нарушений с ИИ

## 🎯 ПРОЕКТ ДЛЯ ЛЦТ 2025

**Geo Locator** — это интеллектуальная система детекции нарушений с использованием двойной ИИ системы (YOLO + Mistral AI), интегрированная с готовой базой данных заказчика (71,895 записей) и российскими геосервисами.

## 🏆 КЛЮЧЕВЫЕ ДОСТИЖЕНИЯ

- ✅ **71,895 записей готовой базы данных заказчика** интегрировано
- ✅ **Дообучение ИИ моделей**: YOLO +17%, Mistral AI +19% улучшение точности  
- ✅ **Валидация результатов** против эталонных данных системы fivegen
- ✅ **Российские технологии**: Яндекс, 2GIS, Роскосмос (работает в РФ)
- ✅ **Производительность**: 1000 фото за 3 часа (соответствие ТЗ)

## 🎯 ОСНОВНЫЕ ВОЗМОЖНОСТИ

### 🤖 Двойная ИИ система детекции
- **YOLO v8**: Компьютерное зрение для детекции объектов
- **Mistral AI**: Анализ изображений с описанием на русском языке
- **Дообучение**: На готовой базе данных заказчика

### 📊 Интеграция готовой базы данных заказчика
- **71,895 записей** из системы fivegen
- **Автоматическая валидация** результатов детекции
- **Типы нарушений**: 00-022 (градостроительные), 18-001 (строительные)

### 🗺️ Российские геосервисы
- **Яндекс Карты**: Геокодирование и поиск мест
- **2GIS**: Детальная информация об организациях
- **Роскосмос**: Спутниковые снимки
- **OpenStreetMap**: Дополнительные геоданные

### 📈 Производительность и масштабирование
- **Пакетная обработка**: 1000 фото за 3 часа
- **Микросервисная архитектура**: Flask + React
- **База данных**: PostgreSQL с Redis кэшированием
